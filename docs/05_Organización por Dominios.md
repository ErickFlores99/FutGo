### Organización por Dominios

Cada carpeta dentro de Models representa un dominio del sistema.

Ejemplo:

Models
│
├── Competencia
├── Equipo
├── Partido
├── Estadistica
├── Usuario
└── Seguridad

Cada dominio agrupa las entidades relacionadas entre sí.

Esto facilita:

Localizar archivos rápidamente.
Mantener cohesión.
Evitar directorios con cientos de modelos.
Escalar el sistema sin reorganizar la estructura.
