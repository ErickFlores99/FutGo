### Introducción
¿Qué es FutGo?

FutGo es una plataforma web para la administración de competencias de fútbol, diseñada para centralizar la organización de torneos desde un panel administrativo y ofrecer un portal público para la consulta de información.

Actualmente el proyecto está orientado a administrar:

Competencias
Categorías
Divisiones
Tipos de competencia

Y está preparado para crecer hacia módulos como:

Equipos
Jugadores
Jornadas
Partidos
Árbitros
Estadísticas
Tabla de posiciones
Portal Público

### Objetivos del proyecto
Centralizar la administración de torneos.
Reducir procesos manuales.
Facilitar el seguimiento de competencias.
Escalar fácilmente mediante módulos independientes.
Mantener una arquitectura limpia y mantenible.

El sistema está dividido en dos ambientes.

• Sitio Público
• Panel Administrativo
