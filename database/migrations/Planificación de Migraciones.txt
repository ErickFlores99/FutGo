Planificación de Migraciones


Migración: Usuarios

Tabla: users
- id
- name -> string
- email -> string (unique)
- email_verified_at -> timestamp (nullable)
- password -> string
- estatus -> tinyInteger()->default(4)
    0 Inactivo,1 Activo,2 Bloqueado,3 Eliminado,4 Pendiente,5 Deshabilitado,6 Desactivado
- rememberToken()
- timestamps()

Tabla: roles
- id
- nombre -> string (unique)
- timestamps()

Tabla: rol_user
- rol_id -> foreignId()->constrained('roles')
- user_id -> foreignId()->constrained('users')
- timestamps()


Migración: Competencia_Catálogos

Tabla: competencia_tipos
- id
- nombre -> string (unique)
- descripcion -> string (nullable)
- timestamps()

Ejemplos: Liga,Torneo,Copa

Tabla: generos
- id
- nombre -> string (unique)
- descripcion -> string (nullable)
- estatus -> tinyInteger()->default(1)
    0 Inactivo,1 Activo
- timestamps()

Ejemplos: Varonil,Femenil,Mixto

Tabla: categorias
- id
- nombre -> string (unique)
- descripcion -> string (nullable)
- edad_minima -> integer (nullable)
- edad_maxima -> integer (nullable)
- estatus -> tinyInteger()->default(1)
    0 Inactivo,1 Activo
- timestamps()

Ejemplos: Infantil,Juvenil,Libre, Veteranos

Tabla: divisiones
- id
- nombre -> string (unique)
- descripcion -> string (nullable)
- estatus -> tinyInteger()->default(1)
    0 Inactivo,1 Activo
- timestamps()

Ejemplos: Primera, Segunda, Tercera, Premier


Migración: Competencias

Tabla: competencias

Representa el torneo principal.
- id
- tipo_competencia_id -> foreignId()->constrained('tipos_competencia')
- administrador_id -> foreignId()->constrained('users')
- nombre -> string
- descripcion -> text (nullable)
- fecha_inicio -> date
- fecha_fin -> date
- es_nocturna -> boolean()->default(false)
- estatus -> tinyInteger()->default(0)
    0 Pendiente,1 Activa,2 Suspendida,3 Finalizada,4 Cancelada
- timestamps()

Tabla: competencia_dias

Días en que se jugará la competencia.
- id
- competencia_id
    -> foreignId()->constrained('competencias')
- dia -> tinyInteger
    1 Lunes,2 Martes,3 Miércoles,4 Jueves,5 Viernes,6 Sábado,7 Domingo
- timestamps()

Tabla: competencia_grupos

Representa cada combinación de Género + Categoría + División.
- id
- competencia_id
    -> foreignId()->constrained('competencias')
- genero_id
    -> foreignId()->constrained('generos')
- categoria_id
    -> foreignId()->constrained('categorias')
- division_id
    -> foreignId()->constrained('divisiones')
- estatus -> tinyInteger()->default(1)
    0 Inactivo,1 Activo,2 Finalizado
- timestamps()
